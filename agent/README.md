# Agent Folder Guidelines
This folder contains the skills, tools, and other information that the AI agent like Cursor, Opencode, Codex, Copilot etc.. will find useful before working with the repository.

### Do:

- Use the cache folder to store any markdown files, skills etc that are required for your AI agent to function properly.
- The cache folder is ignored by git, so you can use it to store any files that are generated by your agent.

### Do not:

- Make changes to any other folder or file inside the agent folder except the cache folder without a clear justification.

### Avoid:

- Using the agent folder as a general-purpose storage for unrelated files or documentation.

### Discuss:

- Other folders inside the agent folder are meant for specific purposes if you make changes to them without prior discussion on GitHub, do not include those changes in your PR otherwise we will be unable to merge them. If you think there is a need to change other files or folders, feel free to discuss it with the maintainters and the community.
